To help users getting started with UMEP, the community is working on setting up tutorials and instructions for different parts of the UMEP tool. The tutorials available are found in the table below.

Note

One essential part when working with geodata in a GIS is to make sure that a common coordinate reference system (CRS) is used, both for the data itself and the current QGIS-project you are working in. For more info, see here. It is strongly recommended to reproject/transform all geodatasets into the same projected coordinate system before any processing starts.
