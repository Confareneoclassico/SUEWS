4.3.5. Snow related parameters¶

SnowInitially¶
Requirement:Optional
Description:Flag for initial snow status [0 or 1]
Configuration:If the model run starts when there is no snow on the ground, set SnowInitially = 0 and the snow-related parameters will be set accordingly. If the model run starts when there is snow on the ground, the following snow-related parameters must be set appropriately. The value of SnowInitially overrides any values provided for the individual snow-related parameters. To prevent SnowInitially from setting the initial conditions, either omit it from the namelist or set to -999. If values are provided individually, they should be consistent the information provided in SUEWS_Snow.txt .
